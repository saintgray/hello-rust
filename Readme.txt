Rust 학습을 위한 Repository 입니다.


-------------------- Getting Started -------------------- 
https://www.rust-lang.org/learn/get-started
1) Download RUSTUP-INIT.EXE
2) Install rustup-init.exe
3) Make new project for Rust 
    - cargo new hello-rust
// If you in windows development environment
4) install toolcahin for windows 
    - rustup toolchain install stable-x86_64-pc-windows-gnu
5) change env for windows
    - rustup default stable-x86_64-pc-windows-gnu
////////////////////////////////////////////
6) run main.rs (cargo run)


-------------------- Compile & Run --------------------
1) move to target directory 
2) compile .rs file (cmd : rustc {filename.rs})
3) check generated .exe file
4) run exe file (cmd : ./{filename})
